##Интерфес без движения предметов, т.е. все управление производится по средством кликов для производительности отрисовки##

### 1.png - исходное изображение ###

### 2.png - инвентарь при игре - быстрый доступ к предметам ###
* Чтобы положить предмет в быстрый доступ: нажать на предмет, а затем - на ячейку
* Чтобы снять предмет из быстрого доступа: нажать на ячейку, а затем - на пустую ячейку инвентаря
* В 2-1.png появляются дополнительные слоты для снаряжения
(при вызове полного инвентаря и клике по броне она автоматически встанет на нужный слот, чтобы снять броню - кликнуть на ее слот)

### 3.png - полный инвентаь, представлен в виде списка###
* Появляется при нажатии на клавижу (нивентарь)
* Однако, для удобства, некоторые ячейки могут быть пропущены, чтобы игрок мог сам сортировать предметы
* Чтобы переместить предмет: нажать на предмет, а затем - на другую ячейку. Содержимое будет поменято местами.

### 4.png - Таблица крафтов ###
* Появляется при нажатии на клавишу (крафты)
* Делится на 2 части:
    * Предметы, которые уже доступны для крафты
    * Предметы, для крафта которых имеется хотя бы один предмет (либо все возможные предметы)
* После крафта предмет попадает в инвентарь

### 5.png - Внешнее взаимодействие ###
* Появляется при взаимодействии с блоком
* Дублирует полный инвентарь (в случае если это сундук),
за исключением быстрых кнопок (внизу) на подобие: "взять все", "положить все" и пр.
* Дублирует таблицу крафтов (в случае если это верстак)
* Для других предметов имеет собственный интерфейс. На 5.png имеет интерфейс печки.
    * Печь имеет собственную вместимость по горючему, исходным продуктам и производным продуктам.

### 6.png - Общая картинка ###
* Различные части интерфейса не перекрывают друг друга, а обновляютсяс только при дествиях
* Можно взаимодействовать с различными частями интерфейса одновременно, например, переметить предмет из сундука в инвентарь
